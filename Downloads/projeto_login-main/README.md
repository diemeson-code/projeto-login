# 🔐 Tela de Login Responsiva

## 👨‍💻 Autor

Projeto desenvolvido por **Diemeson Santos** como prática de desenvolvimento Front-End utilizando HTML5 e CSS3.

---

## 📌 Descrição

Este projeto consiste na criação de uma interface de autenticação (Login) moderna e responsiva.

A página permite que o usuário:

* Insira e-mail
* Insira senha
* Recupere senha
* Acesse o sistema
* Realize login alternativo
* Acesse opção de cadastro

O layout foi desenvolvido com foco em organização visual, centralização dos elementos e boa experiência do usuário.

---

## 🧰 Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* Media Queries (Responsividade)

## 🖥️ Layout

A aplicação foi construída em duas seções:

**Área esquerda**

* Imagem de fundo
* Mensagem de boas-vindas

**Área direita**

* Logo
* Formulário de login
* Link de recuperação de senha
* Botão de acesso
* Login com Google
* Cadastro de usuário

O posicionamento foi realizado com **Flexbox** para manter alinhamento centralizado.

---

## 📱 Responsividade

Para dispositivos móveis foi utilizada Media Query:

* Oculta a imagem lateral
* Centraliza o formulário
* Aumenta os campos para toque
* Adapta para telas pequenas

Assim a aplicação funciona corretamente em smartphones e tablets.
